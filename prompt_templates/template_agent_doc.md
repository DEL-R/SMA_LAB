## Objectif de ce template
Formuler des instructions efficaces à un agent de documentation (Agent_Doc)  
pour créer, reformuler ou maintenir des procédures techniques au format Markdown.

...

```txt
Rédige une procédure Markdown à destination de [cible] pour [objectif].
Inclure : [étapes clés / remarques]. Style : [professionnel / synthétique / détaillé].
```
